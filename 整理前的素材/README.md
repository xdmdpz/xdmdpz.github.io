# 整理前的素材

此目录用于存放博客文章写作前的素材，包括：

- 资料收集
- 草稿
- 参考链接
- 图片素材

## 目录结构

```
整理前的素材/
├── image/              # 图片素材
│   └── 关键词_001.jpg  # 图片命名：去掉日期的md文件名_序号
├── YYYY-MM-DD-关键词.md    # 文章草稿，日期为预计发布日
├── PROMPT.md
└── README.md
```

## 使用说明

1. 将素材放入此目录
2. 图片统一放在 `image/` 文件夹，命名规则：`关键词_序号.jpg`（如 `Agent-Harness_001.jpg`）
3. 文章草稿命名格式：`YYYY-MM-DD-关键词.md`（日期为预计发布日期）
4. 整理后创建正式文章到 `source/_posts/`
